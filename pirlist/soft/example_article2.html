Bloody 7 Mod

Bloody - программа для управления и настройки игровых мышей серии «Bloody Series» от компании A4Tech. Это условно-бесплатное приложение с ограниченным функционалом в бесплатной версии. Bloody предназначена для установки на ПК под управлением Windows 7, 8, 8.1, 10 и 11 (32 и 64 бит). Интерфейс переведён на русский язык.

В чём особенность Bloody 7 Mod?

У данной модификации есть две версии:

  • Bloody 7 ReSkin
  • Bloody Launcher

Bloody 7 ReSkin

Bloody 7 с обновлённым интерфейсом в современном стиле, исправленными визуальными багами и улучшенной читаемостью.

Bloody Launcher

Bloody Launcher позволяет создавать профиль всех настроек Bloody 7 для каждой игры.

В данный лаунчер интегрирован Bloody 7 ReSkin.

Чтобы создать новый профиль, нажмите Add Profile, затем введите желаемое имя профиля и выберите изображение для профиля размером 80×80 пикселей (эта картинка будет отображаться в правом нижнем углу Bloody 7 ReSkin).

ВНИМАНИЕ: изображение должно быть в формате PNG размером 80×80 пикселей, иначе оно может отображаться некорректно!

Картинку можно взять на сайте icons8.ru. При скачивании изображения на этом сайте можно указать размер 80×80 px.
